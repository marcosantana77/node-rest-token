# node-rest-token
Autorization rest token client-server
<br/>
Node<br/>
Express<br/>
Mongoose<br/>
jsonwebtoken<br/>
<br/>
1 - git clone https://github.com/renanbet/node-rest-token.git<br/>
2 - cd server<br/>
3 - npm install<br/>
4 - npm start<br/>
5 - Open client/index.html on browser<br/>
<br/>
see config.js to mongodb configuration
